gitで一部のファイルを特定のコミットに戻すとき

2020-02-11
git

備忘録．なぜか知らないけれど，全然違うファイルを別のファイルに上書き保存してしまったときに調べた．

git log  #戻したいコミット番号を得る
git checkout [コミット番号] [ファイルパス]
