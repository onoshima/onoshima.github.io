Ubuntu 18.04にCiscoAnyconnectを入れたときのメモ

June 18, 2019 (Last Modified: February 22, 2020)
others
Page content

大学のVPN接続にCiscoのソフトウェアを入れる必要があったのだが，大学のマニュアルにはWindows用とMac用のインストール手順しか乗っておらず，ubuntuへの導入については色々と調べる必要があった．結果的に次のサイトで見つけたコードをターミナルに打ち込んだら問題なく導入された．

sudo apt-get update
sudo apt-get install network-manager-openconnect
sudo apt install libpangox-1.0-0
sudo apt-get install lib32z1 lib32ncurses5
cd ~ && mkdir download-anyconnect
cd download-anyconnect && wget http://www.isthe.com/for-you-only/anyconnect/OLD/anyconnect-predeploy-linux-64-4.3.05017-k9.tar.gz
tar -xzvf anyconnect-predeploy-linux-64-4.3.05017-k9.tar.gz
cd /anyconnect-4.3.05017/vpn/
sudo ./vpn_install.sh

sudo systemctl daemon-reload

参照：How to install Cisco Anyconnect on Ubuntu 18.04 64-Bit.
