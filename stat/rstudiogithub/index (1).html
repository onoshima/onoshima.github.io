RstudioでGithubの鍵認証設定
R gitHub Rstudio
Lastmod: 2020-01-25

R studioの鍵認証についての備忘録。またやることになったらおそらく忘れるので。

やったこと

  1. RstudioのToos > Global Option > Git/SVNでView public keyをクリック
  2. 表示される公開鍵をコピー
  3. Githubの自分のページ > Settings > SSH and GPG keysから右上のNew SSH kayというボタンをクリック
  4. 適当な名前をつけて先ほどの公開鍵をペースト
